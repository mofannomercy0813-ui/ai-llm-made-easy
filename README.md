## 这本书是什么

每个想入行大模型的人，都会在某个深夜对着「Self-Attention」「RoPE」「RLHF」这些词发呆。

市面上的教程分两种：一种是「面试八股合集」——给你标准答案但不告诉你为什么。一种是「从零实现教程」——带你写代码但第一章就假设你懂线性代数。

这本书想做第三种：**用一个生活化的比喻开头，用一张手绘图解讲清楚一件事，用一段聊天式的文字让你觉得「哦，原来是这样」**。

## 这本书不是什么

- 不是面试题 Q&A 合集（已有 400-Interview-Questions 等优秀项目）
- 不是从零代码实现教程（LLMs-from-scratch 做得足够好）
- 不是学术综述（不需要引用 50 篇论文）

## 阅读方式

- **零基础**：从头按顺序读，每课的主类比足够让你理解核心概念
- **面试突击**：跳到对应章节，读「追问模块」快速建立面试话术
- **查漏补缺**：每章的「严谨补充」小卡片提供了公式和论文引用

## 文体特色

每课遵循统一模板：

```
开头（感性切入） → 核心内容（图解 + 类比） → 追问模块 → 思考题 → 文化升维
```

我们刻意保持：
- 不写「首先…其次…最后」
- 不用冒号和破折号
- 对话感优先于学术严谨性（严谨内容放进小卡片）
- 每张图都是手工 SVG，不是 AI 生成

## 技术栈

- 文字：Markdown
- 图解：手写 SVG（内嵌）
- 代码示例：Python + PyTorch（可选阅读）
- 静态站点：Python 脚本 → HTML + GitHub Pages
