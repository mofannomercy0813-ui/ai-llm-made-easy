## 为什么有这本书

每个想入行大模型的人，都经历过那个时刻，对着 Self-Attention、RoPE、RLHF 这些词，查了半天资料，还是一头雾水。

市面上的教程，要么是面试八股合集，给你标准答案但不讲为什么。要么是从零代码实现，第一章就默认你数学满分。

这本书想做一件事，**让你在翻开论文之前，脑子里先有一个能跑通的直觉画面。**

Attention 是什么？图书馆查资料的比喻，四步走完。

Transformer 怎么工作的？收费站过车的比喻，一张图讲完。

RLHF 干什么的？脱口秀彩排的比喻，三阶段串通。

每一课都从一个你能瞬间代入的生活场景开始，画一张图，讲一个故事，然后你再看公式，会发现它只是在描述你脑子里已经有的那个画面。

------

## 读完你能做什么

- 用自己的话给一个完全不懂的朋友讲清楚 Transformer
- 面对「RAG 系统怎么设计」「PEFT 方案怎么选」「Agent 怎么拆分」面试题，给出有层次的回答
- 看懂主流大模型论文的核心设计，不再被公式挡在门外

## 阅读方式

不管你是刚入门还是明天面试，都能找到自己的读法。

从头到尾顺着读也行。跳到自己最关心的章节当成词典查也行。面试前翻开最后两课刷真题也行。每课独立，不需要前情提要。
