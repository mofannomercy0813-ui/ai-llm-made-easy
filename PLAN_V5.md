## 新增内容（4 项）

### 1. 解码策略 — 新第 3.5 课（插入 Part 1 世界观）

**为什么必须加**：面试里「Greedy 和 Beam Search 的区别」「Temperature 是干什么的」「Top-K 和 Top-P 哪个好」是基础题。本书 26 课零覆盖。

**插入位置**：第 3 课（Prompt/CoT/ICL）和第 4 课（RLHF 速览）之间，作为 `lesson-03-5.md` 或重排课号。

**内容结构**（~600 字）：
- Greedy Search：每步选概率最高的，快但单调重复
- Beam Search：保留 Top-K 条候选路径，质量高但贵
- Top-K Sampling：从 Top-K 里随机抽，多样性好但 K 难调
- Top-P (Nucleus) Sampling：累积概率超过 P 的 Token 里抽，自适应
- Temperature：控制概率分布平滑度。T<1 确定性高，T>1 更随机
- 一句话选型建议 + 面试怎么答

**状态**：待写

### 2. VLM/多模态简明一课 — 新第 24.5 课（插入 Part 4 落地）

**为什么加**：Datawhale hello-agents 有 13 道 VLM 面试题。2026 面试越来越多问多模态。对不懂 CV 的面试者反而是最容易拉分的知识盲区。

**插入位置**：第 24 课（MoE/GQA/前沿架构）和第 25 课（面试真题）之间。

**内容结构**（~600 字）：
- CLIP 原理：对比学习对齐图文，共享嵌入空间。文本「一只猫」和图片里那只猫的 embedding 距离近
- LLaVA/MiniGPT-4 架构：视觉编码器（CLIP ViT）→ 投影层 → LLM。冻结视觉编码器，只训投影层
- 为什么 VLM 也会幻觉：视觉编码器看错了 → LLM 基于错的输入推理
- 三个面试常见追问 + 怎么答

**状态**：待写

### 3. NLP 基础附录 — 新附录 A

**为什么加**：Happy-LLM 和 dive-into-llms 第一章都是 NLP 基础。本书默认读者有 NLP 背景，但实际读者可能是转行的后端/前端/数据分析。部分课程中「Embedding」「交叉熵」「softmax」等概念对非 NLP 读者不友好。

**插入位置**：全书末尾，第 26 课之后。

**内容结构**（~800 字，极简速查）：
- One-hot → TF-IDF → Word2Vec → BPE 演进
- Embedding 是什么：高维空间里的语义坐标
- 损失函数两句话：交叉熵（分类）、KL 散度（分布差异）
- Softmax / Sigmoid / GELU 什么时候用
- 超参与的概念比：epoch / batch size / learning rate / warmup

**状态**：待写

### 4. 理论速查表 — 新附录 B

**为什么加**：无竞品做这个。面试前 10 分钟翻一遍，覆盖最常被追问的数字和公式。

**插入位置**：附录 A 之后。

**内容结构**（纯表格，~400 字）：
- 显存速算：FP32/FP16/INT8/INT4 各多大
- 计算量速算：C ≈ 6ND、KV Cache 显存、LoRA 参数量
- 常用默认值：r=8, α=16, β=0.1, ε=0.2, T=0.7
- 公式速查：Attention、RoPE、PPO Clip、DPO Loss
- 开源模型速查表：Qwen2.5 / DeepSeek-V3 / Llama 3 参数/架构/特点
