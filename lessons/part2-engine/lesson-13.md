# 第 13 课：两种 Mask，为什么模型必须戴眼镜

Padding Mask：句子不等长时用 PAD 补齐。PAD 位没有语义，不能参与 Attention。掩码位置设为 -∞，过 softmax 后 e^(-∞)=0，权重归零。Encoder 和 Decoder 都需要。

Sequence Mask（因果掩码）：下三角矩阵，对角线以下保留，以上全部屏蔽。Decoder 训练时不能偷看未来，推理时看不到未来，训练时看到了就会产生曝光偏差。只用于 Decoder 的自注意力。Encoder 是双向注意力，不需要。

两个 mask 实际相加。被任一 mask 屏蔽的位置设为 -∞。用 0 不行，e^0=1，被屏蔽的 token 仍会分到权重。只有 -∞ 才能精确归零。

---

## 追问模块

**追问：「掩码值为什么是 -∞ 而非 0？」** e^0=1，被屏蔽的 token 仍能分到注意力。e^(-∞)=0 精确归零。

**追问：「推理时 Sequence Mask 还起作用吗？」** 必须。自回归生成一次一个 token，已有序列逐位增长，Mask 随长度动态扩大。

**追问：「Encoder 要不要 Sequence Mask？」** 不要。Encoder 是双向注意力，需要同时看见前后。
